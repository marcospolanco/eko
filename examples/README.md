# Examples

This directory contains illustrative, testable EKO examples, from a minimal empirical claim through policy resolution and workflow instantiation.

Start with [`seaway_cba/`](seaway_cba/), a source-grounded Collective Bargaining Agreement conversion. It pairs the source document with EKO artifacts, runtime inputs, expected outcomes, and failure or escalation paths. The fixtures specify behavior for a future pinned interpreter; this repository does not include that interpreter.
